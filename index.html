🧪 Lilly Signature Add-in

Test the add-in in Outlook Web Access (OWA)

What is this?

This is a testing page for the Lilly Signature Add-in. This add-in automatically inserts your Lilly email signature when composing new emails in Outlook Web Access (OWA).

Note: This add-in is currently in testing phase. Only authorized Lilly employees can test it.

Features

  • ✅ Auto-insert your professional signature on new emails
  • ✅ Edit signature preferences in the task pane
  • ✅ Quick Insert button for fast signature insertion
  • ✅ Supports both Desktop and Mobile Outlook

How to Test in OWA

Step 1: Open Outlook Web Access

Go to https://outlook.office365.com and sign in with your Lilly credentials.

Step 2: Click the Settings Gear Icon

In the top right corner, click the Settings icon (⚙️) and select "View all Outlook settings".

Step 3: Navigate to Add-ins

Go to Settings → Integrated apps → Add-ins.

Step 4: Get Add-ins

Click "Get Add-ins" button.

Step 5: Upload the Manifest

Look for an option to "Upload an add-in" or "Upload my manifest".

Paste the manifest URL:

https://raw.githubusercontent.com/adityajichakar-lly/signature-addin-assets/main/signature_manifest_github_test.xml
Step 6: Add the Add-in

Click "Add" to add the add-in to your OWA.

Step 7: Test It Out

Create a new email. You should see:

  • A "Lilly Signature Manager" group in the ribbon
  • "Quick Insert" button - inserts your signature instantly
  • "Edit Signature" button - opens the signature editor pane

Troubleshooting

Issue: "Cannot install the add-in" error

This usually means the manifest URL is incorrect or unreachable. Verify the URL in your browser first.

Issue: Buttons don't appear in the ribbon

Try refreshing OWA (Ctrl+F5 or Cmd+Shift+R). Sometimes the cache needs to be cleared.

Issue: Signature doesn't auto-insert

Make sure you're composing a new message (not replying). The auto-insert only works for new emails.
